Vue Bootstrap Sidenav MDB Pro组件

Vue Sidenav-Bootstrap 4和材料设计

Vue Bootstrap sidenav是一种垂直导航组件，除了传统的文本链接之外，还可以嵌入图标，下拉菜单，头像或搜索表单。

由于其清晰和简单，它显着提高了用户体验。它使您可以快速浏览小型应用程序和大型门户。多重链接嵌入功能使您能够实施更高级的内容分类，这在大型项目中几乎是必不可少的。

借助MDB，您可以通过使用各种诱人的侧边菜单之一轻松地在自己的项目中实现SideNav。

基本用法

修身侧面导航

采用瘦财产 mdb-side-nav 组件以创建边栏的精简版并添加其他内容 mdb-side-nav-item 与拨动开关属性以最小化菜单。在您的数据中，创建指示侧边栏当前状态的属性，并将其与已崩溃物业 mdb-side-nav 和 mdb-side-nav-item 。有关更多详细信息和选项，请参阅本节的“ API”选项卡。
